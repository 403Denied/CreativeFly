# Restored Creative Fly [Client]
This mod allows you to fly faster in creative mode.

- You can toggle fast flight using a keybind (default 'J')
- You can change flight speed with mouse wheel by holding a keybind (default 'J')
- You can use [Mod Menu](https://modrinth.com/mod/modmenu) to change flight speed
---
**Only use on servers where you have permission to use this mod or you risk getting banned.**

# Restored Creative Fly [Client][RU]
Этот мод позволит вам летать быстрее в креативе

- Вы можете включить быстрый полёт горячей клавишей (по умолчанию 'J')
- Вы можете изменить скорость быстрого полёта колёсиком мышки с зажатой горячей клавишей (по умолчанию 'J')
- Вы можете использовать [Mod Menu](https://modrinth.com/mod/modmenu) чтобы изменить скорость полёта
---
**Используйте только на серверах, где у вас есть разрешение на использование этого мода, иначе вы рискуете получить бан.**



This mod is based on and uses imagery from the original [Creative Fly](https://modrinth.com/mod/creative-fly) mod by [TheEvilM](https://modrinth.com/user/TheEvilM). This mod is simply an update to the original mod, and will be discontinued if or when the original becomes active again.
